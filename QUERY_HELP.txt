CXP Query Command - Help Output
================================

$ cxp query --help

Search for a term in CXP file contents

Usage: cxp query [OPTIONS] <FILE> <TERM>

Arguments:
  <FILE>  CXP file to search
  <TERM>  Search term or pattern

Options:
  -l, --limit <LIMIT>      Maximum number of results to return [default: 10]
  -c, --context <CONTEXT>  Lines of context around matches [default: 2]
  -v, --verbose            Verbose output
  -h, --help              Print help


Example Commands:
-----------------

1. Basic search:
   $ cxp query project.cxp "function"

2. Search with more results:
   $ cxp query project.cxp "TODO" --limit 20

3. Search with more context:
   $ cxp query project.cxp "error" --context 5

4. Full options:
   $ cxp query project.cxp "async fn" --limit 50 --context 3 --verbose


Main CXP Commands:
------------------

$ cxp --help

CXP - Universal AI Context Format

Usage: cxp [OPTIONS] <COMMAND>

Commands:
  build    Build a CXP file from a directory
  info     Show information about a CXP file
  list     List files in a CXP archive
  extract  Extract a file from a CXP archive
  query    Search for a term in CXP file contents
  help     Print this message or the help of the given subcommand(s)

Options:
  -v, --verbose  Verbose output
  -h, --help     Print help
  -V, --version  Print version
